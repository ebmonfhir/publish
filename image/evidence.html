14.10 Resource Evidence - Content

Clinical Decision Support Work GroupMaturity Level: 0 Trial UseSecurity Category: Business Compartments: Not linked to any defined compartments

This represents statistics, certainty, both the intended and actual population, and evidence variables.

14.10.1 Scope and Usage

14.10.2 References

This resource is referenced by Group.

This resource does not implement any patterns.
